# Documentation française — CollectiveAccess + plugin museesDeFrance

Documentation technique et fonctionnelle, en français, pour utilisateurs et administrateurs de **CollectiveAccess** et du **plugin museesDeFrance**.

## CollectiveAccess

[CollectiveAccess](https://collectiveaccess.org) est une suite open source d'applications de gestion et de publication de collections destinée aux musées, archives, bibliothèques spécialisées et fondations patrimoniales. Le projet a été fondé en 2003 par [Whirl-i-Gig](https://github.com/collectiveaccess) à Brooklyn et est distribué sous **licence GNU GPL v3**.

CollectiveAccess est référencé sur le **SILL** (Socle Interministériel des Logiciels Libres, [ID 498](https://code.gouv.fr/sill/detail?id=498)) depuis le 16 mai 2024, sur Wikidata sous l'[item Q2982932](https://www.wikidata.org/wiki/Q2982932), et a sa [fiche Wikipedia FR](https://fr.wikipedia.org/wiki/CollectiveAccess).

## Plugin museesDeFrance

[IdéesCulture](https://www.ideesculture.com) (SAS française basée au Mans depuis 2012) a développé et maintient le plugin **museesDeFrance**, **officiellement validé par le Service des Musées de France** :

- **Registres d'inventaire** : validation [décembre 2013](https://www.culture.gouv.fr/thematiques/musees/pour-les-professionnels/conserver-et-gerer-les-collections/informatiser-les-collections-d-un-musee-de-france/informatisation-reglementaire-des-collections-d-un-musee-de-france/procedure-de-validation-des-fonctionnalites-d-edition-informatisee-des-registres-d-inventaire-et-de-depots-des-outils-informatiques-des-musees-de-f)
- **Récolement décennal** : validation [janvier 2014](https://www.culture.gouv.fr/thematiques/musees/pour-les-professionnels/conserver-et-gerer-les-collections/informatiser-les-collections-d-un-musee-de-france/informatisation-reglementaire-des-collections-d-un-musee-de-france/procedure-de-validation-des-fonctionnalites-liees-au-recolement-decennal-des-collections-des-musees-de-france)
- **Export Joconde** : en cours de validation, premier déploiement au Musée du Château de Mayenne

Le plugin et sa documentation sont publiés sous licence libre. Le repository [museesDeFrance](https://github.com/ideesculture/museesDeFrance) contient le code source.

## À propos d'IdéesCulture

IdéesCulture intervient en France et dans l'espace francophone sur tous les volets d'un projet CollectiveAccess :

- **Accompagnement métier** : modélisation documentaire, paramétrage, conduite de projet
- **Hébergement souverain** : infrastructure en France (OVHcloud Roubaix · Gravelines · Strasbourg), RGPD-natif
- **Migration de données** : reprise depuis FileMaker, Excel, AtoM, Omeka, Micromusée, Flora, etc.
- **Formation** : utilisateurs et administrateurs
- **Développement** : ~70 dépôts open source publics maintenus

107 institutions accompagnées depuis 2012, dont des musées labellisés *Musée de France*, l'INRAP, plusieurs DRAC, le ministère de la Culture (Mobilier National), 5 ministères français au total, et des institutions belges, suisses, canadiennes, mahoraises.
